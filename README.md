# java-admin-system

一个用于 Java 后端转岗练习的权限管理后台系统（接口版）。

## 技术栈
- Java 17
- Spring Boot 3.x
- MyBatis-Plus
- MySQL 8
- Redis 7
- Sa-Token
- Knife4j
- Docker

## 本地启动
1. 启动 MySQL
2. 启动 Redis
3. 运行项目启动类
4. 打开接口文档
